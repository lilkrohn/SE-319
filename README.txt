To compile and run my programs, follow the instructions for each task below:

For task 1: 
- Open the snake.html file in browser
- To start the snake game, click start
- Use the "Turn Left" and "Turn Right" buttons to change the snake's direction
- If the snake hits a wall, an alert will pop up and the game will stop

For task 2: 
- Read-line sync must be installed in order for the node.js code to compile
- Navigate to the correct directory and run the "hw3.js" file in the terminal
- Hit enter after each number you type, and enter again to load the output
